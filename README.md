# MatrixShop

MatrixShop 是 Matrix 系列中的综合商店插件，覆盖系统商店、玩家商店、全球市场、拍卖、交易、购物车、记录查询和箱子商店等核心交易场景。

当前 `codex/bootstrap-v1` 分支是主线功能之外的开发分支，用于持续推进命令结构、模块入口和交互细节的演进。

## 分支说明

- 面向 MatrixShop 的迭代开发与结构调整
- 保持与主线功能一致的 Matrix 系列风格
- 用于承接功能整合与命令路由优化

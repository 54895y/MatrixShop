## English Section 🇬🇧

### Introduction
Welcome to the MatrixShop! This project aims to provide a comprehensive collection of matrix operations in various languages including Python, Java, and C++.

### Features
- Matrix addition
- Matrix multiplication
- Determinant calculation
- Eigenvalue decomposition

### Quick Start
1. Clone the repository:
   ```bash
   git clone https://github.com/54895y/MatrixShop.git
   ```
2. Install dependencies:
   ```bash
   cd MatrixShop
   pip install -r requirements.txt
   ```

# MatrixShop

MatrixShop 是 Matrix 系列中的综合商店插件，面向服务器内的系统商店、玩家商店、全球市场、拍卖、交易、购物车、交易记录和箱子商店等场景。

当前 `main` 分支用于维护 MatrixShop 的主线代码，面向正式部署与持续迭代。

## 功能定位

- 系统商店与菜单入口
- 玩家商店、全球市场和拍卖体系
- 交易、购物车与交易记录
- 模块化配置与多入口命令路由
- JDBC 优先、文件回退的数据层设计
